Ecommerce Website

Technologies Used: HTML5, CSS, Bootstrap, JS, APIs

I used cutting-edge web technologies to create a smooth user experience while developing a fully working e-commerce website. The project uses HTML, CSS, Bootstrap, and JavaScript to provide an interactive "Add to Cart" system with login and signup capabilities. In addition, this project emphasises responsive design techniques, which guarantee a user-friendly experience on all platforms. A colour palette tool for altering UI elements, modal-based forms for simplified interactions, and dynamic product listings are some of the key features. The website offers a strong platform for online shopping and places equal emphasis on utility and aesthetics.
